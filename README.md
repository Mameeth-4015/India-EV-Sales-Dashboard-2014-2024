## 🚀 Overview

India's EV market has undergone a **dramatic transformation** from 2014 to 2024. This Power BI dashboard captures every milestone of that journey — from early adoption through policy-driven surges to mainstream penetration. Whether you're a policy researcher, market analyst, or EV enthusiast, this dashboard gives you a comprehensive, data-driven lens into India's green mobility story.

---

## 📊 What's Inside

| Module | Description |
|--------|-------------|
| 🗓️ **Year-on-Year Trends** | Annual EV sales volume from 2014 to 2024 with growth rate overlays |
| 🏍️ **Segment Breakdown** | Two-wheelers, three-wheelers, four-wheelers, buses & commercial EVs |
| 🗺️ **State-Wise Analysis** | Regional adoption heatmaps across all Indian states & UTs |
| 🏭 **Manufacturer Insights** | Top OEM market share and competitive landscape |
| 📈 **Market Penetration** | EV share as % of total vehicle sales over time |
| ⚡ **Policy Impact Tracker** | FAME I, FAME II & PLI scheme correlation with sales spikes |

## ⚡ Key Insights at a Glance

```
🔋  10x Growth     →  EV sales grew over 10× between 2019 and 2024

🏭  OEM Giants     →  Ola Electric, TVS & Bajaj lead the 2W segment
```

## 🚀 Getting Started

### Prerequisites
- [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (Free) — version **March 2024 or later** recommended

### Installation

```bash
# 1. Clone this repository
git clone https://github.com/your-username/India-EV-Sales-Dashboard.git

# 2. Navigate to the project folder
cd India-EV-Sales-Dashboard

# 3. Open the dashboard
# → Double-click India_EV_SaLes_Dashboard_2014-2024.pbix
# → Or open via Power BI Desktop → File → Open
```

### Refreshing Data *(optional)*
If you have updated source data:
1. Go to **Home → Transform Data → Data Source Settings**
2. Update file paths to your local data
3. Click **Refresh** to reload
